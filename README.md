# pxec
Execute **scripts** through aliases

# Building
``cargo build``

# Platforms
``Linux``

# How to use
| input            | example          | explanation                                      |   |     |
|------------------|------------------|--------------------------------------------------|---|-----|
| run alias        | pxc email        | execute <alias>                                  |   |     |
| run builtin cmd  | pxc ls           | list all entries                                 |   |     |
